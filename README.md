# Next.js Carousel API Project

A **Next.js** project that displays a carousel with images fetched from an API.  
The fifth image in the carousel repeats the first image.

## Features

- Next.js App Router
- API route returning 4 images
- Carousel displaying 4 images + 5th repeats 1st
- Fully responsive and interactive

## Tech Stack

- Next.js 13+
- React 18+
- CSS / Tailwind (optional)

## Getting Started

```bash
npm install
npm run dev
